# Praktikum Pemrograman Web Dasar
Repository ini berisi kumpulan tugas dari mata kuliah **Praktikum Pemrograman Web Dasar**.

### Tugas 1
Membuat beberapa halaman web menggunakan **HTML dan CSS**, yaitu:
* Halaman **Index**
* Halaman **Form**
* Halaman **Landing**
* Halaman **Profile**

### Tugas 2
Melanjutkan **Tugas 1** dengan menambahkan **PHP** pada halaman **Index, Landing, dan Profile**.
Pada tugas ini menerapkan konsep dasar PHP, yaitu variabel, array, perulangan, dan percabangan.
Data yang diinput melalui **form** akan diproses dan ditampilkan kembali secara **dinamis menggunakan PHP**.
